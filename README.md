# Exlent-IoT-Week

## Install Arduino IDE

This repository includes a shell script named `install_arduino.sh` to download and install the Arduino IDE AppImage on Linux.

1. Download the `install_arduino.sh` file.

2. Open a terminal.

3. Go to your Downloads folder:

```bash
cd ~/Downloads
```

4. Give the installer script permission to run:

```bash
chmod +x install_arduino.sh
```

5. Run the installer:

```bash
./install_arduino.sh
```

The script downloads the Arduino IDE into:

```bash
~/Applications/ArduinoIDE.AppImage
```

After the installation finishes, you can open Arduino IDE with:

```bash
~/Applications/ArduinoIDE.AppImage
```
